# Crypto Lab 2：非对称加密与后量子密码

大家好，欢迎来到「Crypto 专题一：非对称加密与后量子密码」的实验部分！本节 Lab 分为必做题目和选做题目。

以下是一些提示：

- 理论部分的课件见 [Crypto 专题一：非对称加密与后量子密码](https://courses.zjusec.com/slides/crypto-lec2-v1.pdf)；课上介绍了一些针对常见密码学算法的常见攻击方式，能够将它们加以应用即可；
- 经测试，DeepSeek V4 Pro、GLM 5.2 等常见的 LLM 可以顺利完成以下题目，因此若完成实验的过程中遇到困难，可以优先考虑向 AI Agent 求教；若无果，欢迎向助教寻求帮助；
- 分数可能会根据实际情况进行调整，无需过于担心 Lab 成绩的问题，将重点放在理解课程内容上、尽力而为即可，相信大家可以收获满意的体验与结果。

祝玩得开心！:P
